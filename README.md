# BNG Integration Allocation

This is an Azure Function for Developer Registration. 

## Prerequisites

- Node.js (version 16+ recommended)
- npm (comes with Node.js)
- An Azure Functions account


## Installation

First, clone this repository to your local machine:

```bash
git clone https://github.com/DEFRA/bng-integration-allocation
```
Navigate to the cloned repository:

```bash
cd bng-integration-allocation
npm install
npm start
```
Unit testing
```bash
npm run test
```
